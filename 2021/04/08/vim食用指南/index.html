vim食用指南

发表于2021-04-08|更新于2021-04-08
|阅读量:

vim是世界上最强大的文本编辑器，即使他的学习曲线十分陡峭。(vim的格言Better, Stronger, Faster.) 下面这篇博文是一些vim技>能的汇编(其实就是东拼西凑QAQ)将尽快教会你vim，但是你也需要更多的练习来真正掌握它们。

1.基础

1.1 模式

vim有Normal模式，也就是刚进入的模式，命令模式。
当你按下i，你会进入Insert模式，这就是你编辑文本的插入模式，当你想要回到命令>模式，请按下Esc。

在Normal模式下的推荐记住的基础命令

  • x -> 删除当前光标所在的一个字符
  • :wq -> 保存并退出
  • dd -> 删除当前行，并且把删除内容存入剪贴板
  • p/P -> 粘贴剪切板，小写后，大写前
  • yy -> 相当于ddP
  • hjkl -> 左下上右←↓↑→
  • :help -> 显示各种命令的帮助,按:q退出帮助

    1.2 更多在Normal模式的命令

  • 各种插入
  • a -> 在光标后插入
  • o/O -> 在当前行后/前插入新行
  • cw -> 替换从光标所在位置后到一个单词结尾的字符
  • 移动光标
  • 0 -> 行头
  • g_ -> 到本行最后一个不是空格、tab等的位置
  • /pattern -> 搜索字符串，pattern被替换
  • Undo/Redo
  • u -> undo
  • ctrl + r -> redo
  • 文件操作
